Le Point De Départ

Description :

Situé au 48, rue Solférino, Le Point de Départ dispose à la fois d'un bar et d'un restaurant. Cet endroit est idéal pour boire un verre ensemble entre amis/collègues ou pour passer du bon moment familial. L'établissement accepte la réservation et les cartes de crédits. Le Point de Départ est ouvert tous les jours de 15:00 à 23:30 sauf le samedi et le dimanche.
